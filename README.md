# API Server

Simple Rest API using gin(framework) & gorm(orm)

## Endpoint list

### Emails Resource

```
GET    /emails
GET    /emails/:id
POST   /emails
PUT    /emails/:id
DELETE /emails/:id
```

### Users Resource

```
GET    /users
GET    /users/:id
POST   /users
PUT    /users/:id
DELETE /users/:id
```

server runs at http://localhost:8080
